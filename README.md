## Upload to GitHub

1. Open your GitHub repository: hde-community-directory.
2. Click **uploading an existing file**.
3. Open this ZIP on your computer and drag **the contents inside the folder**, not the ZIP itself, into GitHub:
   - index.html
   - .nojekyll
   - README.md
4. Click **Commit changes**.

## Turn on GitHub Pages

1. Go to **Settings** in the repository.
2. Click **Pages** on the left.
3. Under **Build and deployment**, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / root
4. Click **Save**.
5. Wait a few minutes. GitHub will show the live website link.

## Important privacy note

On a free GitHub account, GitHub Pages generally requires a public repository. The password screen keeps casual visitors out, but it is not the same as true account-based security.
